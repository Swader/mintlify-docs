### Overview

- **name**: IbvbWBTC
- **path**: `vb/tokens/IbvbWBTC.sol`
- **context**: vaultbridge
- **tags**: vaultbridge, token, wbtc, bridge, destination
- **title**: IbvbWBTC
- **notice**: Interface for the Bridged Vault Bridge WBTC token. This is deployed on the destination chain - for "Katana" this means Katana, for "Tatara" and "Bokuto" this means these respective testnets. The address for each context is different, and indicated in custom tags.
